# View the 'CDC PLACES' data dictionary

This function provides the user with a data frame that shows all of the
available measures in the PLACES data set and for which release years
the measures are included.

## Usage

``` r
get_dictionary()
```

## Value

a dataframe with the current PLACES data dictionary.

## Examples

``` r
# First save the data
if (FALSE) { # \dontrun{
dictionary <- get_dictionary()
} # }
# Then view the data frame
# View(dictionary)
```
